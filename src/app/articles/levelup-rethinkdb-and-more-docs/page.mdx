## A New RethinkDB Adapter

We have [a common test suite](https://github.com/feathersjs/feathers-service-tests) that we run against all the database adapters so we are certain that it complies with the [common query syntax](http://docs.feathersjs.com/databases/querying.html) and has the same interface as the other adapters, however since RethinkDB has this fancy real-time feature called ‚Äúchange feeds‚Äù, if you use this new adapter you get **real time all the way from the front to the back!**
